## Overview

mwP Сoub releases updates regularly to add new features, fix bugs, and improve performance. You receive notifications via email and the dashboard at `https://dashboard.example.com`. Check this changelog to see what's new.

<Callout kind="info">
  Subscribe to our RSS feed or GitHub releases to stay informed about new versions.
</Callout>

## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration for documentation pages, allowing multiple users to edit simultaneously.
- Introduced webhook support for custom integrations. Configure at `https://dashboard.example.com/webhooks`.

## Improvements

- Enhanced search performance by `>50%` for large documentation spaces.
- Updated UI with brand color `#3B82F6` for better visibility.

## Bug Fixes

- Fixed rendering issues with nested lists in MDX previews.
- Resolved authentication errors when using `{API_KEY}` with special characters.

**Backward Compatibility:** All changes are non-breaking. Update via `npm install mwp-сoub@latest`.

</Update>

## How to Upgrade

Follow these steps to upgrade mwP Сoub safely.

<Steps>
  <Step title="Check Current Version" icon="package">

    Run the version check:

````bash
npx mwp-сoub --version
````

  </Step>
  <Step title="Backup Data" icon="database">

    Export your documentation:

    ```bash
    mwp-сoub export --output=backup.json
    ```

  </Step>
  <Step title="Install Latest" icon="download">

    <CodeGroup tabs="npm,yarn">

````bash
npm install mwp-сoub@latest
````

````bash
yarn add mwp-сoub@latest
````

    </CodeGroup>

  </Step>
  <Step title="Migrate Config" icon="settings">

    Update your config file if needed:

````json
{
  "apiUrl": "https://api.example.com",
  "apiKey": "YOUR_API_KEY"
}
````

  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first. View full migration notes in the [release notes](https://github.com/mwp-сoub/releases).
</Callout>
