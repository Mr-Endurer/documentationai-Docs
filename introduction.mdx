## Overview

mwP Сoub helps you organize and manage all your project documentation in dedicated spaces. Create custom documentation hubs for teams, products, or clients with version control, search, and collaboration tools. You streamline knowledge sharing and reduce onboarding time across your projects.

## Key Features

mwP Сoub offers powerful tools to handle documentation at scale. Explore the core capabilities below.

<Columns cols={3}>

## Get Started in Minutes

Follow these steps to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.mwpcoub.com` and create your account with email or GitHub.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" in the dashboard. Name it, like "Project Alpha Docs", and set visibility.
  </Step>
  <Step title="Add Your First Doc" icon="file-text">
    Upload Markdown files or write directly. Use the editor for rich formatting.

````mdx
# Welcome to Project Alpha

This is your first document.
````

  </Step>
  <Step title="Invite Team" icon="mail">
    Share the space link or add members via email. Set roles like Editor or Viewer.
  </Step>
</Steps>
